# Edu-Web-MANTRA-2026

Motive Educations

Motive Educations is a responsive frontend website created using HTML and CSS. It offers a simple and organized platform where users can learn about courses, explore educational content, and navigate through different sections with ease.

PROJECT DESCRIPTION
This project was completed as part of the MANTRA 2026 program to practice frontend web development concepts, responsive web design, and basic SEO techniques.

Features

- Responsive layout
- Home page
- About page
- Courses page
- Gallery page
- Contact page
- Registration form
- Easy-to-use navigation
- SEO-friendly design
- Fast page performance


TECHNOLOGIES USED

- HTML5
- CSS3
- Netlify (Hosting & Deployment)

SEO Features

- Semantic HTML elements
- Responsive web design
- Optimized page titles
- Meta description
- Image "alt" attributes
- Search engine friendly structure
- Fast loading pages
